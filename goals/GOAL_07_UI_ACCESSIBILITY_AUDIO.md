# Goal 07 — Complete UI, Accessibility, and Audio

## Objective

Finish all release UI and inclusive settings.

## Work

- Main, pause, collection, traits, settings, results, credits.
- Korean and English localization.
- Reduced motion, shake, contrast, color presets, large HUD.
- Volume buses and mute.
- Stage-based audio stems.
- Focus management and keyboard menu navigation.
- Safe-area handling.
- Contextual tutorial.

## Acceptance criteria

- Menus usable without pointer.
- 200% text scaling does not clip core actions.
- Color is not the only relationship signal.
- All audio can be muted.
- No unsafe flashing.
- Localization has no missing production keys.
