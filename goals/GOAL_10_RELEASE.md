# Goal 10 — Release Candidate

## Objective

Produce a verifiable production release candidate.

## Work

- Full regression.
- Browser/device matrix.
- Save migration and recovery drill.
- 30-minute desktop and 20-minute mobile soak.
- Dependency/license report.
- CSP and host headers.
- Privacy page.
- Accessibility review.
- PWA install/update test.
- Final clean-room originality review.
- Tag and release notes.

## Acceptance criteria

Every item in `docs/14_RELEASE_CHECKLIST.md` is checked or has a documented, explicitly accepted exception. No unresolved P0/P1 defect. Release artifact is reproducible from the tagged commit.
