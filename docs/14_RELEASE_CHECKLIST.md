# Release Checklist

## A. Product

- [ ] First absorption occurs naturally within 5 seconds in the default seed range.
- [ ] First expansion is understandable without tutorial text.
- [ ] Edible, contested, and threat states are distinguishable without color alone.
- [ ] Settlement is available and explained after stage 1.
- [ ] Death still grants the documented reduced reward.
- [ ] No paid random reward exists.
- [ ] Launch content count meets the GDD.

## B. Originality and licensing

- [ ] No reference source code or transformed code exists in history.
- [ ] UI, names, art, palettes, sounds, and constants passed originality review.
- [ ] Repository license is present.
- [ ] Third-party notices are complete.
- [ ] All asset provenance records are complete.
- [ ] Dependency licenses were reviewed.

## C. Functional

- [ ] Mouse control.
- [ ] Keyboard control.
- [ ] Touch direct control.
- [ ] Touch joystick option.
- [ ] Pause/resume.
- [ ] Background/foreground lifecycle.
- [ ] Portrait/landscape resize.
- [ ] Voluntary settlement.
- [ ] Death settlement.
- [ ] Unlock purchase and equip.
- [ ] Settings persistence.
- [ ] Korean/English switch.
- [ ] Offline launch after first visit.
- [ ] Safe service worker update.

## D. Accessibility

- [ ] Reduced motion.
- [ ] Camera shake toggle.
- [ ] High contrast relationship mode.
- [ ] Color vision presets.
- [ ] 200% UI text scale.
- [ ] 44 px targets.
- [ ] Keyboard navigation for menus.
- [ ] Visible focus.
- [ ] Full mute.
- [ ] No unsafe flashing.

## E. Persistence

- [ ] Fresh profile.
- [ ] Save after settlement.
- [ ] Backup rotation.
- [ ] Corrupted primary recovery.
- [ ] Both invalid recovery screen.
- [ ] Migration from every released version.
- [ ] Future version refusal.
- [ ] Export/import validation.
- [ ] No silent currency loss.

## F. Performance

- [ ] P95 frame time target met on test devices.
- [ ] Entity caps respected.
- [ ] Particle pool bounded.
- [ ] 30-minute desktop soak.
- [ ] 20-minute mobile soak.
- [ ] No unbounded heap growth.
- [ ] DPR cap verified.
- [ ] Adaptive quality does not oscillate.
- [ ] Bundle and initial transfer budgets pass.

## G. Security and privacy

- [ ] No runtime third-party requests.
- [ ] CSP configured.
- [ ] No unsafe HTML injection.
- [ ] Import fuzz tests pass.
- [ ] Service worker caches same-origin known assets only.
- [ ] No sensitive native permissions.
- [ ] Privacy page matches actual behavior.
- [ ] Store data declarations match actual behavior.

## H. CI/CD

- [ ] Locked install.
- [ ] Typecheck.
- [ ] Lint.
- [ ] Unit tests.
- [ ] Coverage gate.
- [ ] Production build.
- [ ] E2E smoke.
- [ ] Artifact size check.
- [ ] License report.
- [ ] Preview deployment.
- [ ] Tagged production release.
- [ ] Rollback/forward-fix procedure documented.

## I. Store assets

- [ ] App icon.
- [ ] Adaptive Android icon.
- [ ] Feature graphic where required.
- [ ] Phone screenshots.
- [ ] Tablet screenshots.
- [ ] Localized description.
- [ ] Privacy policy.
- [ ] Support contact.
- [ ] Age rating.
- [ ] Release notes.
